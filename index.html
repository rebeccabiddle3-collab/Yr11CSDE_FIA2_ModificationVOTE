The original experiment

Craik, Govoni, Naveh-Benjamin & Anderson (1996) found that divided attention during encoding causes greater memory impairment than during retrieval.

Three conditions (class prac)

Control — full attention
Encoding distraction — reaction task while hearing words
Retrieval distraction — reaction task while recalling

What you are doing

You are modifying this experiment — refining, extending or redirecting it to address your own related hypothesis. The class agrees on one shared change.

Does the emotional content of words affect how much divided attention hurts memory?

Replace the standard word list with two matched lists — one of emotionally charged words (e.g. grief, danger, love, terror) and one of emotionally neutral words (e.g. table, pencil, window, folder). All three attention conditions are kept. Compare recall across word types.

Does same-modality distraction cause more memory interference than cross-modality distraction?

Keep the word list and three attention conditions, but compare two different secondary tasks: one group performs an auditory task (respond when you hear a target tone) and another performs a visual task (press a key when a shape changes on screen). Which disrupts auditory word memory more?

New IVSecondary task modality — auditory vs visual reaction task
DV (kept)Words recalled; reaction time on the secondary task (seconds)
PredictionAn auditory secondary task will cause greater memory impairment than a visual task when words are presented auditorily — same-modality tasks compete for the same cognitive resource
TheoryBaddeley's Working Memory Model — the phonological loop handles auditory information; two auditory tasks compete for the same limited resource, causing greater interference
Feasibility✅ Manageable — auditory tone apps and visual shape-change tasks both free online; requires some preparation of two secondary task versions

Does divided attention disrupt free recall more than recognition memory?

Keep all three attention conditions, but change how memory is tested. Half the class uses free recall (write down every word you can remember) and the other half uses a recognition task (circle the words you saw from a list of 30, where 15 are targets and 15 are distractors). Compare how much divided attention impairs each test type.

New IVMemory test type — free recall vs recognition
DVFree recall: words correctly recalled; Recognition: hits minus false alarms
PredictionFree recall will show a larger divided attention impairment than recognition — because free recall requires effortful, controlled retrieval, while recognition is more automatic and cue-supported
TheoryControlled vs automatic processes (Lesson 7.2) — free recall is a controlled process requiring selective attention; recognition can draw on automatic familiarity signals that are less disrupted by a secondary task
Feasibility✅ Very easy — recognition sheet prepared in advance; same word list; straightforward scoring
